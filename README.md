# 📊 Sales & Revenue Analysis Dashboard

## 📌 Project Overview

The Sales & Revenue Analysis Dashboard is a Business Intelligence project developed using Microsoft Power BI. It provides an interactive dashboard to analyze sales performance, revenue trends, product performance, and regional sales distribution. The dashboard helps businesses monitor key performance indicators (KPIs) and make data-driven decisions.

---

## 🎯 Objectives

- Analyze sales and revenue data.
- Track business performance using KPIs.
- Identify top-performing products.
- Compare revenue across different regions.
- Visualize sales trends over time.
- Enable interactive filtering using slicers.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI Desktop
- Microsoft Excel
- Power Query
- Data Visualization

## 📊 Dataset Information

The dataset contains **100 sales records** with the following fields:

- Date
- Product
- Category
- Region
- Quantity
- Sales
- Revenue

---

## 📈 Dashboard Features

### KPI Cards

- Total Sales
- Total Revenue
- Total Quantity
- Average Revenue

### Visualizations

- Revenue by Product (Bar Chart)
- Sales by Category (Column Chart)
- Revenue by Region (Pie Chart)
- Sales Trend (Line Chart)

### Interactive Slicers

- Product
- Category
- Region
- Date

---

## 🚀 How to Run the Project

1. Open Microsoft Power BI Desktop.
2. Import **Sales_Revenue_Dataset_100.xlsx**.
3. Create KPI cards and charts.
4. Add Product, Category, Region, and Date slicers.
5. Save the report as **Sales_Revenue_Analysis_Dashboard.pbix**.
